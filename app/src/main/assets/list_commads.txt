// <- так можно комментировать
//
// список команд:
// change phrase <Имя персонажа> <Фраза> <- смена фразы
// Если указать "none" в поле "имя", то имя персонажа исчезнет, словно это мысли рассказчика.
//
// change bg <Название фона в ресурсах> <- смена фона
//
// change person <Имя персонажа> <Эмоция> at <Позиция> <- смена персонажа
//
// show person <Позиция> <- Плавно выводит персонажа на сцену
//
// hide person <Позиция> <- Плавно убирает персонажа со сцены
//
// move person <Позиция> to front/back <- перемещение персонажа на передний или задний план
//
// play music/sound <Название аудио в ресурсах> <- воиспроизведение музыки или звуков
// stop music <- остановка воспроизведение музыки
//
// update scene <- обновляет внесённые объяснения
